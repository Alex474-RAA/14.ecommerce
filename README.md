# 14.ecommerce - Ядро интернет-магазина

    Проект реализует базовые классы для интернет-магазина
    на Python с использованием ООП.


## Функциональность

    - **Product**: товар с названием, описанием, ценой и количеством
    - **Category**: категория товаров с автоматическим подсчетом
    - Атрибуты класса для статистики магазина

## Запуск

# Установка зависимостей
    poetry install

# Запуск программы
    python src/main.py

# Запуск тестов
    pytest tests/ -v

# Проверка кодстайла
    flake8 src/ tests/
